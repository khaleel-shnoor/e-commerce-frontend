# Setup Guide

## Prerequisites

- Node.js 18+
- npm

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit `http://localhost:5173`

## Build

```bash
npm run build
npm run preview
```

## Demo Logins

Use the demo buttons on the login page:

- **Customer** → `/account`
- **Seller** → `/seller`
- **Admin** → `/admin`

## Stack

- React 19 + Vite
- React Router DOM
- Tailwind CSS v4
- Lucide React icons
- Context API (auth, cart, theme, toast)
